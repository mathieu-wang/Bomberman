Detailed Description

This class is a widget that displays the Main menu.

It includes the following buttons that the user can interact with:
playButton: emit playGameSignal when clicked.
logoutButton: emit logoutGameSignal when clicked.
loadButton: emit loadMenuSignal when clicked.
quitButton: emit quitGameSignal when clicked.
showLeaderboardButton: emit showLeaderboardSignal when clicked. changeSettingsButton: emit changeSettingsSignal when clicked
