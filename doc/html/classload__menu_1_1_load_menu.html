Detailed Description

This class is a widget that displays the Load menu.

It includes the following buttons and fields that the user can interact with:
loadButton: emit loadSavedGameSignal when clicked.
returnButton: emit backSignal when clicked.
savedGameList: list of all the saved game of the current active user.
