Detailed Description

This class is a widget that displays the pause menu.

It includes the following buttons that the user can interact with:
resumeButton: emit resumeGameSignal when clicked.
saveButton: emit saveMenuSignal when clicked.
laodButton: emit loadMenuSignal when clicked.
showLeaderboardButton: emit showLeaderboardSignal when clicked.
backButton: emit backToMainMenuSignal when clicked.
quitButton: emit quitGameSignal when clicked.
