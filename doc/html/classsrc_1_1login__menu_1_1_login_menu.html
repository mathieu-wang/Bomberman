Detailed Description

This class is a widget that displays the login menu.

It includes the following buttons and fields that the user can interact with:
youName: is a LineEdit widget the user uses to enter his name.
username: is a LineEdit widget the user uses to enter his username at login or signup.
password: is a LineEdit widget the user uses to enter his password at login or signup.
signupButton: calls register() method when clicked.
loginButton: calls login() method when clicked.
